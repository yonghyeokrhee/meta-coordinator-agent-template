### 8.3 응대 규칙 (CN 특화)

1. **이중 언어 응답 권장**. CN 채널 문의에는 가능하면 **중국어 + 한국어 두 언어로** 답변 (KR AM 검토 가능하도록). 사내 응대용일 때는 한국어만으로 충분.
2. **자동화 X 항목** (총 95건):
 - 결제 - 신용카드 거절 / 알리페이 미표시 / 인증 미수신 / 사이트 접속 불가 / 紫鸟 브라우저 / ASIN 검색 안됨 등
 - → 자동 답변이 어렵고 휴먼 검토가 필요한 영역. CS Agent 는 **"확인 후 안내드리겠습니다"** 응답 후 사람 에스컬레이션
3. **자동화 O 항목** (총 255건, 72.9%):
 - 본 문서의 한국어 답변을 그대로 활용 가능
 - 답변 본문 확정은 다른 KB 문서 (02 FAQ · 04 Help Center) 의 정의를 인용
4. **CN 특수 토픽** (다른 KB 에 없는 항목):
 - `紫鸟浏览器` (지냐오 브라우저) 화이트리스트
 - 法人 vs 个人 支付宝 (법인 vs 개인 알리페이)
 - 多店铺关联风险 (다중 스토어 계정 연관)
 - 对公账户 (법인 계좌이체) 및 发票
 - → 이 항목들은 **본 문서가 1차 출처**. 답변 본문은 별도 정리 필요 (현재는 질문만 정제됨)
5. **민감 정보 주의**. 다른 고객의 ASIN, 계정명, 매출 수치는 응답에 포함하지 않음 (다른 KB 와 동일 규칙).
