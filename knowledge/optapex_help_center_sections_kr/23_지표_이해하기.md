# 23. 지표 이해하기

대시보드 제공 Metrics 항목 설명표

## Product Report

### Total Performance

| 항목 | 의미 | 설명 |
|------|------|------|
| Total Sales | 총 매출 | 기간 내 발생한 오가닉 매출 + 광고 매출 |
| Estimated Fee | (예상) 수수료 | 기간 내 발생 매출에 대한 예상 아마존 수수료 → 주문 처리 수수료(FBA Fee) + 판매 수수료(Referral Fee) + 환불 수수료(Return Fee) + 보관 수수료(Storage Fee) |
| Estimated Margin | (예상) 마진 | (총 매출 - 광고 비용 - 예상 Amazon 비용) / 총 매출 |
| Pageviews | 총 클릭수 | 기간 내 발생한 클릭 개수 (광고 클릭 포함) |
| Sessions | 고유 클릭수 | 기간 내 발생한 고유 클릭(방문자) 개수 |
| Buy Box % | 바이박스 획득 비율 | 바이박스를 획득하고 있는 시점의 조회 수를 상품 상세 페이지의 전체 조회 수로 나눈 값 |
| CVR(Total) | 총 전환율 | 기간 내 구매 전환율 (광고 전환 포함) |
| Units Sold | 총 판매 수량 | 기간 내 제품 판매 개수 (오가닉 판매 수량 + 광고 판매 수량) |
| Units Returned | 환불 수량 | 기간 내 발생한 환불 제품 개수 (* 환불 날짜가 아닌 환불된 제품의 구매 날짜 기준으로 집계됨) |
| Sales Returned | 환불 금액 | 기간 내 발생한 환불된 매출액 |

### Sponsored Products (SP)

| Metric | Definition |
|--------|------------|
| SP Spend | 기간 내 발생한 총 광고 비용 |
| SP Sales | 기간 내 발생한 광고 매출 (Same SKU + Other SKU) |
| SP ROAS | SP Sales ÷ SP Spend |
| SP Units Sold | 기간 내 광고를 통해 발생한 판매 수량 |
| SP Sales (Same SKU) | 기간 내 발생한 동일 SKU 광고 매출 |
| SP Units Sold (Same SKU) | 기간 내 광고를 통해 발생한 동일 SKU 판매 수량 |
| SP ACOS | SP Spend ÷ SP Sales |
| SP Impressions | 기간 내 광고 노출 |
| SP Clicks | 기간 내 광고 클릭 횟수 |
| SP CTR | 기간 내 광고의 클릭률, SP Clicks ÷ SP Impressions |
| SP Avg. CPC | 기간 내 평균 광고 클릭 당 비용, Spend ÷ SP Clicks |
| SP CVR | 기간 내 광고 구매 전환율, SP Units Sold ÷ SP Clicks |

### Sponsored Display (SD)

| Metric | Definition |
|--------|------------|
| SD Spend | 기간 내 발생한 총 광고 비용 |
| SD Sales | 기간 내 발생한 광고 매출 |
| SD Promoted Click Sales | 광고 ASIN에서 발생한 광고 매출 |
| SD Units Sold | 기간 내 광고를 통해 발생한 판매 수량 |
| SD Clicks | 기간 내 광고 클릭 횟수 |
| SD Impressions | 기간 내 광고 노출 횟수 |
| SD Impressions Views | Viewable placement에 게재된 SD 광고의 노출 수 |
| SD Cumulative Reach | SD 광고에 노출된 고유 Reach 수 |
| SD Detail Page Views | SD 광고 클릭 이후 발생한 상품 상세 페이지 조회수 |
| SD New to Brand Detail Page Views | 최근 12개월간 구매 이력이 없는 신규 고객(New-to-Brand)의 상품 상세 페이지 조회수 |
| SD New to Brand Sales | 신규 고객(New-to-Brand)에 의해 발생한 매출 |
| SD ROAS | SD Sales ÷ SD Spend |
| SD ACOS | SD Spend ÷ SD Sales |
| SD CVR | SD Units Sold ÷ SD Clicks |
| SD View CTR | 조회된 노출(Viewable Impression) 대비 클릭 전환율 |
| SD New to Brand Sales Rate | 신규 고객(New-to-Brand) 매출 비중(%) |
| SD Impression Frequency Avg. | SD 광고 1인당 평균 노출 횟수 |
| SD Viewability Rate | 뷰어블(Viewable) 기준을 충족한 SD 광고 노출 비율(%) |
| SD New to Brand Detail Page View Rate | 신규 고객(New-to-Brand) 유입 상세 페이지 조회 비율(%) |

> ⚠️ **중요 참고사항**
> - 일부 Amazon 수수료(폐기 수수료, 청산 수수료, 제거 수수료, 딜 수수료, 세금 등)는 추정 마진 계산에 포함되지 않습니다.
> - 광고 매출 및 광고비에는 **Sponsored Products(SP)**와 **Sponsored Display(SD)**만 포함되며, Sponsored Brand 및 Sponsored TV는 현재 지원되지 않습니다.

## Brand Report

### Sponsored Brands (SB)

| Metric | Definition |
|--------|------------|
| Impressions | 기간 내 광고가 노출된 총 횟수 |
| Clicks | 기간 내 광고를 클릭한 총 횟수 |
| CTR | 노출 대비 클릭 전환율. Clicks ÷ Impressions × 100 |
| Brand Searches | 캠페인에서 프로모션된 브랜드를 검색한 횟수로, 광고 인터랙션(클릭 또는 노출)에 기인하여 집계 |
| Detail Page Views | 광고 클릭 이후 발생한 상품 상세 페이지(PDP) 조회 수 |
| 5-second views Video | 비디오 광고를 5초 이상 시청한 노출 횟수(5초 미만에서 종료된 경우 미집계) |
| Complete views Video | 비디오 광고를 끝까지 시청한 노출 횟수(이벤트는 노출당 1회만 집계되며 재시청은 포함되지 않음) |
| Complete views (video ad) Video | 비디오 광고 소재 기준 완전 시청 횟수(Video AD Type 선택 시에만 유효한 데이터가 표시됨) |
| VTR Video | 노출 대비 비디오 완전 시청 비율(Complete views ÷ Impressions × 100) |
| Ad Spend | 선택 기간 내 Sponsored Brands 캠페인에 지출한 총 광고비 |
| CPC | 클릭 1회당 평균 광고 비용(Ad Spend ÷ Clicks) |
| CPM | 노출 1,000회당 광고 비용 (Ad Spend ÷ Impressions × 1,000) |
| VCPM | 조회 가능한 노출(Viewable Impression) 1,000회당 비용. 광고의 최소 50%가 화면에 1초 이상 노출된 경우에만 Viewable Impression으로 집계. |
| Purchases | 광고를 통해 발생한 총 구매 건수 (Transaction) (1 Purchase = 1 거래) |
| Units | 광고를 통해 판매된 총 상품 수량 |
| Ad Sales | 선택한 Attribution Window 내에서 Sponsored Brands 광고를 통해 발생한 총 판매액 |
| ROAS | 광고 수익률(Ad Sales ÷ Ad Spend) |
| ACOS | 판매 광고 비용 비율 (Ad Spend ÷ Ad Sales × 100) |
| NTB Purchases NTB | 최근 365일 내 해당 브랜드에서 구매 이력이 없는 신규 고객의 구매 건수 |
| NTB Purchases (%) NTB | 전체 Purchases 중 NTB Purchases가 차지하는 비율(NTB Purchases ÷ Purchases × 100) |
| NTB Sales NTB | 신규 브랜드 고객(New-to-Brand)에 의해 발생한 총 판매액 |
| NTB Sales (%) NTB | 전체 Ad Sales 중 NTB Sales가 차지하는 비율(NTB Sales ÷ Ad Sales × 100) |
| Cost per Purchase (NTB) NTB | 신규 브랜드 고객 1명 획득 비용(Ad Spend ÷ NTB Purchases) |
