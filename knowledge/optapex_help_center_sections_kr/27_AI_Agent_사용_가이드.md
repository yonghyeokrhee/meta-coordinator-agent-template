# 27. AI Agent 사용 가이드

> **Note:** 본 기능은 현재 **Beta 버전**으로 운영 중이며, 지속적으로 고도화될 예정입니다.

## 🚀 Optapex AI Agent

**"데이터 확인 → 원인 분석 → 상품 분류 → 최적화 세트 생성까지, AI가 한 번에."**

수많은 지표와 광고 데이터, 어디서부터 확인해야 할지 고민되셨나요?

Optapex AI Agent 광고/판매 지표를 빠르게 요약하고, 변화의 원인을 찾고, 무엇을 바꿔야할지 다음 액션(최적화 SET 생성/변경)을 제안하고 실행합니다.

## 🔍 핵심 기능

### 1) 광고/판매 지표 분석

계정의 주요 KPI를 빠르게 확인하고 이상 신호를 탐지합니다.

### 2) 상품 분류

성과 기반으로 상품(ASIN)을 그룹화하고 우선순위를 제안합니다.

### 3) 실적 변화 원인 분석

캠페인/광고/상품/키워드 단위로 성과 변화의 원인을 구조적으로 설명합니다.

### 4) 최적화 세트 생성/변경

분석 결과를 바탕으로 최적화 세트를 새롭게 생성하거나 수정합니다.

> 📢 솔루션 사용 중 궁금한 점이 있다면 Agent에게 바로 물어보세요. Help Center를 직접 검색하지 않아도, Agent와의 대화만으로 필요한 내용을 빠르게 확인할 수 있습니다.

## AI Agent 사용 방법

### Step 1. 언어 설정하기

언어 설정은 화면 우측 상단에서 변경할 수 있으며, 선택한 언어를 기준으로 AI Agent가 응답합니다.

### Step 2. AI Agent 접속하기

1. Optapex 로그인 > 우측 하단 **플로팅 버튼(💬)** 클릭 > 상단 탭에서 **AI Agent** 선택합니다.
2. 대화 시작 전 분석을 진행할 **Account** 및 **Marketplace**를 선택합니다.

### Step 3. 질문하기

데이터 분석, 상품(ASIN) 분류, 최적화 전략 추천 등 궁금한 내용을 AI Agent에게 먼저 질문해 보세요!

- 대화 진행 중간에 ✔ 확인(Confirm), ⏹ 종료(Stop), Next step과 같은 선택 옵션이 버튼 형태로 제공됩니다.

### Step 4. 최적화 SET 생성/수정하기

1. **상품 분류 확인**: 최적화 SET 구성을 확인하고, 필요 시 조건을 수정합니다.
2. **생성 방식 선택**: 기존 최적화 SET을 수정 또는 신규 생성 중 선택합니다.
3. **예산 및 최적화 SET 설정 검토**
4. **최적화 set 생성**: 최종 확인 이후 최적화 생성 버튼을 클릭하면 실제 적용이 완료됩니다.

### Step 5. History(대화 기록) 확인

- 좌측 하단 **History** 클릭 시 이전 AI Agent 대화 이력을 전체 확인할 수 있습니다.

> ⚠️ 현재 대화 기록 다운로드는 지원하지 않으며, 추후 관련 업데이트가 있을 경우 공지 예정입니다.

## FAQ

**Q. 모든 계정에서 AI Agent를 사용할 수 있나요?**
AI Agent 기능은 구독 중인 전체 계정 대상으로 제공됩니다. 이후 서비스 운영 정책에 따라 **LITE / PRO 플랜 티어 별 사용량 제한**이 적용될 수 있습니다.

**Q. 언어 설정을 변경하고 싶습니다.**
우측 상단 언어 설정에서 한국어, 영어, 중국어 등 원하는 언어로 변경할 수 있습니다.

**Q. 리포트는 어떻게 다운로드하나요?**
광고/판매 실적 분석, 상품 분류 등 분석 결과 리포트는 Agent 채팅창 내에서 **CSV**와 **Markdown, HTML** 형태로 다운로드할 수 있습니다.

**Q. AI Agent는 어떤 데이터를 기준으로 분석하나요?**
AI Agent는 Optapex에 연동된 **Amazon Ads** 및 **Seller Central / Vendor Central** 데이터를 기반으로 분석합니다.
