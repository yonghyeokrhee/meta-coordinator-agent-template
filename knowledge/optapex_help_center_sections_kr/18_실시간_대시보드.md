# 18. 실시간 대시보드

## 실시간 대시보드 이해하기

성과를 매 순간 확인하고, 즉각적으로 대응하세요.

Real-Time Dashboard에서는 성과를 시간 단위로 추적하고, 변화 흐름을 즉시 포착하며, 예산 집행 속도를 적절히 유지할 수 있습니다.

---

## 실시간 대시보드 구성

실시간 대시보드는 **세 가지 주요 영역**으로 구성되어 있습니다.

### Overall Performance (전체 성과 요약)

이 영역은 현재 시점까지 누적된 성과를 한눈에 보여주는 요약 화면입니다.

주요 기능:
1. **Optimization Set 선택** - 대시보드 전체를 선택한 Optimization Set 기준으로 필터링합니다.
2. **새로고침 버튼** - 대시보드의 모든 데이터를 최신 시점으로 갱신합니다.
3. **최신 데이터 시점** - 가장 최근에 반영된 데이터의 기준 시간을 표시합니다.
4. **성과 요약 (Performance Overview)**
5. **정렬 기준 지표 (Sorting Metric)**
6. **상품 점유율 차트 (Product Share Chart)**

### Real Time Performance

이 영역에서는 매출, 광고비, 주요 지표를 시간 단위로 확인할 수 있습니다. 가로 스크롤을 통해 최대 **7일간**의 시간대별 성과를 조회할 수 있습니다.

### Inventory Pacing & Budget Pacing

재고 소진 속도와 예산 집행 흐름을 동시에 관리하세요.

---

## 실시간 데이터 이해하기

- **데이터 보정**: Amazon은 무효 트래픽 제거 및 데이터 보정을 위해 최대 72시간이 걸릴 수 있습니다.
- **반품 반영 방식**: 실시간 매출 수치는 이미 반품이 차감된 값입니다.
- **데이터 갱신 주기**: 일반적으로 **1시간 단위**로 업데이트됩니다.
