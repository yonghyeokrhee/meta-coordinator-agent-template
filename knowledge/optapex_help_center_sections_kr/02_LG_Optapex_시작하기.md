# 2. LG Optapex 시작하기

Amazon 계정을 Optapex에 연결하고 최적화 세트를 생성하는 방법을 안내합니다.

LG Optapex 도입 절차는 간단합니다. 계정을 연결하고 첫 번째 최적화 세트를 설정하면 바로 이용할 수 있습니다.

## Step 1 - Amazon 계정 연결하기

LG Optapex를 이용하려면 **Amazon Ads Console 계정**과 **Amazon Seller Central 또는 Vendor Central 계정**을 모두 연결해야 합니다.

- **[데이터 연동하기 → 섹션 3](#3-데이터-연동하기)**: Ads Console 계정 연결 후 Seller/Vendor Central을 순차적으로 연동합니다. 연동 완료만으로는 데이터 수집이 시작되지 않으며, Trial 활성화 후 수집이 시작됩니다.
- **[Seller/Vendor Central 계정 연결하기 → 섹션 3](#3-데이터-연동하기)**: `[↔ Product Data]` 버튼으로 계정 유형(Seller/Vendor)과 지역을 선택해 연동합니다. 연결 완료 시 보라색·파란색 박스로 상태가 표시됩니다.
- **[지원 지역 및 국가 안내 → 섹션 3](#3-데이터-연동하기)**: 북미/남미(미국·캐나다·멕시코·브라질), 유럽(EU 전 지역), 아시아 태평양(일본·호주)을 지원합니다.
- **[여러 계정 추가하기 → 섹션 3](#3-데이터-연동하기)**: 동일한 연결 절차를 반복해 복수 계정을 추가할 수 있으며, 연결 가능한 계정 수에 제한이 없습니다. 구독료는 마켓플레이스 단위로 부과됩니다.

## Step 2 - 30일 Trial 활성화하기

계정 연결이 완료되면 모든 Pro 기능이 포함된 **30일 무료 체험**을 활성화할 수 있습니다. 광고 데이터가 완전히 수집되기까지는 최대 **72시간**이 소요됩니다.

## Step 3 - 최적화 세트 설정하기

데이터 수집이 완료되면 최적화 세트를 설정하여 LG Optapex의 최적화 엔진을 실행할 수 있습니다.

- **[최적화 기본 개념 → 섹션 11](#11-최적화-세트-설정하기)**: 최적화 세트는 동일한 예산·목표·설정을 공유하는 ASIN 그룹 단위입니다. Optapex가 세트 내 ASIN별 성과를 비교해 예산을 자동 배분합니다.
- **[최적화 세트 생성 5단계 → 섹션 11](#11-최적화-세트-설정하기)**: ① ASIN 추가 → ② 최적화 목표 및 고급 옵션 선택 → ③ 광고 상품(SP/SD) 선택 → ④ 기간·예산·CPC 가드 설정 → ⑤ 캠페인 생성 및 키워드 연동.
- **[Ads Console에서 Optapex 캠페인 구조 확인하기 → 섹션 11](#11-최적화-세트-설정하기)**: Optapex가 생성한 캠페인은 `"mop"` 접두어로 시작하며, Ads Console에서 포트폴리오 단위로 확인할 수 있습니다.

## Step 4 - 최적화 세트 모니터링 및 개선

적절한 시점에 최적화 세트를 조정하면 광고 성과를 크게 개선할 수 있습니다. 최적화 세트를 모니터링하고 조정하는 방법을 안내합니다.

- **[최적화 라이프사이클 → 섹션 10](#10-최적화-라이프사이클)**: Week 1~2 학습 기간(Amazon Auto) → Week 3~4 초기 최적화 → Week 5~8 본격 최적화 순으로 진행됩니다. 완전한 최적화까지는 약 2개월이 소요됩니다.
- **[예산 모니터링 → 섹션 12](#12-최적화-세트-모니터링)**: Budget Usage 경고 신호를 확인하고, 예산 시뮬레이터로 현재 예산이 목표 달성에 충분한지 사전에 검토합니다.
