# meta-coordinator-agent-template

A GitHub-ready OpenClaw agent workspace template for CS and engineering issue triage.

This template is for teams that want an agent that:
- turns raw customer support input into structured triage
- builds an issue skeleton first
- classifies category and severity conservatively
- recommends likely modules and owners
- handles no-response follow-up without falsely resolving cases
- keeps durable handling history in either an issue tracker or local logs
